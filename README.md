# Student-Course-Management-System-SQL
Student Course Management System: A MySQL-based database implementation featuring structured schema design, data integrity constraints, and sample data management.

# Project Goal

Designed and implemented a relational database for a student management system to apply core SQL concepts like schema design, table constraints, and data manipulation.  

# Key Technical Implementations

# Database Architecture: 
Developed student_course_db with a focus on data normalization and integrity.  

# Schema Constraints: 
Implemented PRIMARY KEY for unique student identification and UNIQUE constraints for contact information like emails and phone numbers.  

# Data Types: 
Utilized appropriate MySQL data types including BIGINT for phone numbers and DATETIME for registration logs to ensure storage efficiency.  

# Validation: 
Verified the system through complex SELECT queries to ensure all 20 sample records were correctly mapped and stored.
